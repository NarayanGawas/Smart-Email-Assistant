# 📧 Smart Email Assistance

Smart Email Assistance is an AI-powered tool that helps users generate professional and context-aware email replies instantly. It improves productivity by saving time and ensuring well-structured communication.

---

## 🚀 Features

* ✨ Generate email replies using AI
* 🎯 Choose tone (Professional, Friendly, Casual)
* ⚡ Fast and real-time response generation
* 📩 Works with email content input
* 🧠 Enhances communication quality

---

## 🛠️ Tech Stack

**Frontend:**

* React.js
* Material UI

**Backend:**

* Java (Spring Boot)

**Other Tools:**

* REST API
* Git & GitHub
